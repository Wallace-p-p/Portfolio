    "Computational Program for the Identification of Celestial Objects by means of Light Curve Analysis"

    Obs: This is a "scientific initiation" project, mentored by Dr. ZALEWSKI, Willian. Project done in the period 01/08/2017 - 31/07/2018.

    Resume:

    Variable stars are one of the main sources of information for astronomy, the study of the variation of brightness given in light curves, among other data, can provide information about their system, such as whether it has exoplanets or not. In this work, we analyze 36 systems with exoplanets identified from observations made by the NASA Kepler Space Telescope. In this analysis we look for patterns in the brightness variation that repeat in these systems that may be related to the existence of exoplanets. The patterns sought are morphological patterns present in the light curves of each system, the data are analyzed in time series forms, using techniques to find motifs, enabling temporal analysis of the phenomenon.

    Repository on Github.

    Keywords: astronomy, time series, clustering, machine learning.

    "AUTOMATIC IDENTIFICATION OF EXONS AND INTRONS REGIONS IN DNA SEQUENCES"

    Obs: This is a "scientific initiation" project, mentored by Dr. ZALEWSKI, Willian. Project done in the period 01/08/2018 - 31/07/2019.

    Resume:

    The study of DNA sequencing enables the development of new methods to diagnose diseases and the formulation of new drugs. Protein-coding genes constitute the functional part of DNA, where the basic biological instructions for gene expression (protein production) are located, and are composed of intron segments, non-coding sequences, and exons, sequences that will be translated into proteins. Advances in the automation of DNA sequencing have enabled the creation of large amounts of data, generating a high demand for more efficient methods of analysis. Computational techniques such as machine learning have been widely used, but the overall performance of these methods is still not considered satisfactory. In this work we use an aggregate of machine learning algorithms to identify the nucleotides that best identify the subsequence in order to identify intron and exon regions.

    Github Repository

    Keywords: machine learning, dna, classification, sequences.

    "APPLICATION OF MACHINE LEARNING TECHNIQUES FOR PATTERN IDENTIFICATION IN DNA SEQUENCES"

    Obs: This is a "scientific initiation" project, mentored by Dr. ZALEWSKI, Willian. Project done in the period 05/08/2019 - 31/10/2020.

    Resume:

    Advances in the automation of DNA sequencing have enabled the creation of large amounts of data, generating a high demand for more efficient analysis methods. Computational techniques such as machine learning (ML) offer the possibility of processing large amounts of data through various forms of analysis. In this context, in this work we aim to use the classifier Ensemble technique to combine different predictors and a weight system for the identification of introns and exons regions of gene sequences.

    To achieve this goal we employ the use of the Python 3.0 programming language, through which we use the AM algorithms made freely available by the scikit-learn library together with our own lines of code. The method proposed in this paper consists in combining classifiers by means of the voting technique from a set of single-hypothesis predictors. Each predictor classifies a given sequence by assigning a weight, based on the classifier's performance on the training set.

    To validate the proposed method we worked with the Molecular Biology (Splice-junction Gene Sequences) Data-Set database, which contains 3190 DNA sequences with 60 nucleotides each, classified into which type of junction they are intron-exon (IE) (767 sequences), exon-intron (EI) (768 sequences) or none (NE) (1655 sequences). As a result of the experimental evaluation, we could obtain an increase in accuracy in the performance of the voter we built (with classification accuracy of 91% for EI class, 92% for IE class and 94% for NE class), compared to the classifier that does not use the idea of custom weights provided by the library used (with classification accuracy of 93% (EI), 90% (IE) and 88% (NE)).

    Considering the experimental evaluation performed in this study , we observe that our approach demonstrated interesting results for the studied dataset. The use of different prediction techniques combined for classification enabled higher performance in terms of the accuracy rate. More specifically, it is noteworthy that the combination of predictors obtained better results than any of the predictors evaluated individually.

    Github Repository

    Keywords: machine learning, dna, programming, classification, sequences.
